# Sistema de Biblioteca em Java

Projeto simples desenvolvido em Java utilizando Programação Orientada a Objetos.

## Objetivo

Simular um pequeno sistema de biblioteca onde é possível:

- Criar livros
- Adicionar livros em uma biblioteca
- Listar livros cadastrados

##  Tech Utilizadas

- Java
- VS Code
- Programação Orientada a Objetos
- ArrayList

## Estrutura do projeto

Livro.java → representa um livro  
Biblioteca.java → gerencia os livros  
Main.java → executa o programa

## Exemplo de saída

Dom Casmurro - Machado de Assis (1899)
1984 - George Orwell (1949)
